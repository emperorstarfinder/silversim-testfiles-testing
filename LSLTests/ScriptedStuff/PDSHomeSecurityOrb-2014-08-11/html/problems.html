Problems Updating?

Version 12.008+

No known problems

Version 9 to 12.007

Stand near the HS Orb and type this command before trying to update again. This will point these Orbs to the new update server. You will not see a response from the Orb.

/8server cedd82b1-e1ff-b45f-b35f-d232583c0fac

Pre-Version 9

If you wish to upgrade you will need to trade in your old PDS HS Orb for the latest version. The upgrade will cost 100L. IM Psyke Phaeton.

Installation Problems

  • Both the PDS HS Orb and the land MUST technically have the same owner.
  • If the land is deeded you must deed the PDS HS Orb to the same group unless using Push.
  • Resetting the scripts will break them. The whole PDS HS Orb may need replacing.
  • You will need to be within 20m for the Orb to hear you and for you to see it's messages. You can also shout at it when within 100m but you might be too far away for the reply.
  • Check that 'Up' and 'Down' are not both set to zero by using /8status and look for Up and Down. Fix it with /8up=500 and /8down=500 commands.
  • When turning it on choose TurnOn from the menu and then select one of the options.

Known SecondLife Problems/Bugs:

  • Push is useless beyond about 10m since v1.20 Havok4 Region updates. Don't use push for anything but tiny areas.
  • Version 9 RemoteDisplays no longer work due to SL changes. Use Version 8.
  • There is a Second Life "bug" that causes avatars in adjacent regions to be detected only once every 6 seconds.
  • Cross Region border information is unreliable. Attempts have been made to compensate but your mileage may vary.
  • The HS Orb will not recognise you across a Region border because it can't recognise you as an avatar rather than an object.
  • When using 'Group Safe' and 'Group Operated' Second Life will report that the HS Orb and avatars are in the same group if both have no group set.
  • Using the update server may be slow or time out. Try again later.
